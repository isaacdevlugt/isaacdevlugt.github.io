Current Research

Currently, my main interests are in quantum Monte Carlo (QMC) simulations for Rydberg atoms. Using QMC, I hope to provide high-quality numerics for Rydberg atom phases on arbitrary lattices using finite-size scaling analysis. Not only this, but QMC can also be used as input data for generative models. In this way, we can obtain wavefunction representations of the QMC.

Papers

My publications involve quantum Monte Carlo simulations, open-source software for quantum state tomography using neural networks, RBMs for state reconstruction, and ab. initio studies on transition-metal boron-halide/fluoride structures. Check out my google scholar to learn more.
